# 3.4. Participações - Padrões de Projeto

Para garantir a participação de todos os membros, dividimos nosso grupo em equipes, cada uma focada em tarefas específicas. Uma equipe foi responsável pela elaboração dos diagramas Criacional, Estrutural e Comportamental, enquanto a outra se encarregou da implementação dos códigos correspondentes a esses diagramas. Após a conclusão inicial, cada equipe revisou o trabalho da outra, garantindo assim a qualidade e a coesão do projeto.
